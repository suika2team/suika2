Multilingualization

In a story file, you can write a message with a language specifier. 

+en+This is an English message.
+ja+日本語のメッセージです。

The same rule applies to lines with characters' names. 

+en+*Hina*This is an English message.
+ja+*ひな*日本語のメッセージです。

+en+ means a message will be shown only if the system locale is English.

Available languages are the following.

ID Language
ja Japanese
en English
fr French
it Italian
es Spanish
de Deutsch
el Greek
ru Russian
zh Chinese (Simplified)
tw Chinese (Traditional)
other Other languages

If you wish to make a language selection while running a visual novel, write: 

@config locale.force en

This will force-set the current language.
