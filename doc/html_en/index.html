OpenNovel Manual

OpenNovel is a tool to create visual novels.

Usage (Windows)

  • Copy the game.exe file to the sample/ folder.
  • In the sample/ folder, double click the game-win.exe file.

Usage (macOS)

  • Open the tools/game-mac.dmg file then copy the Game app to the sample/ folder.
  • In the sample/ folder, double click the Game app.
