OpenNovel マニュアル

OpenNovel はビジュアルノベルを制作するためのツールです。

使い方

  • game.exe を sample の中にコピーします
  • sample の中で game.exe を起動します

主要ドキュメント

  • 入門編はこちらです
  • 命令仕様はこちらです
  • よくある質問はこちらです

Android 版の作り方

  • export-kit/android/app/src/main/assets の中にゲームの内容をコピーします
  • Android Studioで開きます
  • APKファイルをGoogleドライブで共有すれば配布できます

iOS 版の作り方

  • Apple Developer Programへの加入(1万3千円/年)しておきます
  • XcodeでApple IDを使ってログインしておきます
  • pack.exe で package.pak を作成します
  • export-kit/ios/Resources の中に package.pak をコピーします
  • export-kit/ios/Resources/mov の中に MP4 ファイルをコピーします (なければスキップ)
  • Xcodeで開きます
  • IPAファイルをAirDropでiPhoneに送ればインストールできます

    • Unity 版の作り方

    • export-kit/unity/Assets/StreamingResource の中にゲームの内容をコピーします
    • ゲーム機のSDKで make コマンドを実行します
    • ゲーム機用の Unity で開きます
