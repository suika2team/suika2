OpenNovel マニュアル

OpenNovel はビジュアルノベルを制作するためのツールです。

使い方 (Windows)

  • game-win.exe を sample の中にコピーします
  • sample の中で game-win.exe を起動します

使い方 (macOS)

  • game-mac.dmg を開いて、中の Game を sample の中にコピーします
  • sample の中で Game を起動します

Android アプリの作り方

  • export-kit/android/app/src/main/assets の中にゲームの内容をコピーします
  • Android Studioで開きます
  • メニューから Build App Bundle(s) / APK(s) --> Build APK(s) をクリックします
  • 右下に表示されるバナーから Locate を選ぶと、app-debug.apk が表示されます
  • APK ファイルをGoogleドライブなどで共有すれば配布できます

iOS アプリの作り方

  • 有償の Apple Developer Program に加入しておきます (必須)
  • Xcode で Apple ID を使ってログインしておきます
  • pack-win.exe で package.pak を作成します
  • export-kit/ios/Resources の中に package.pak をコピーします
  • export-kit/ios/Resources/mov の中に MP4 ファイルをコピーします (なければスキップ)
  • Xcode で開きます
  • iPhone を Mac に接続します
  • Xcode からアプリを実行します

Windows アプリの作り方

  • pack-win.exe で package.pak を作成します
  • package.pak と game-win.exe を ZIP 圧縮して配布します

macOS アプリの作り方

  • project.txt で release=yes とします
  • pack-win.exe で package.pak を作成します
  • export-kit/macos/Resources の中に package.pak をコピーします
  • export-kit/macos/Resources/video の中に MP4 ファイルをコピーします (なければスキップ)
  • Xcode で開きます
  • Xcode からアプリを実行します

Unity アプリの作り方

  • export-kit/unity/Assets/StreamingAssets の中にゲームの内容をコピーします
  • PS5/Xbox/Switch の開発ツールで make コマンドを実行します (Windows/Mac の場合は何もしない)
  • Unity で開きます
  • Player Settings で　Allow unsafe code にチェックします
  • アセット内の MainScene をダブルクリックします
  • ウィンドウ左側の Script を OpenNovelScript.cs に結び付けます
  • ウィンドウ左側の BGM, SE, Voice, SYSSE をそれぞれ OpenNovelAudio.cs に結び付けます
  • 実行します
