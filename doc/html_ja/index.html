OpenNovel マニュアル

OpenNovel はビジュアルノベルを制作するためのツールです。

使い方 (Windows)

  • game.exe を sample/ の中にコピーします
  • sample/ の中で game.exe を起動します

使い方 (macOS)

  • tools/game-mac.dmg を開いて、中の Game を sample/ の中にコピーします
  • sample/ の中で Game を起動します
