OpenNovel マニュアル

OpenNovel はビジュアルノベルを制作するためのツールです。

使い方 (統合エディタ版)

  • editor.exe を起動します
  • メニューの ファイル -> 新規ゲームを作成 -> 日本語アドベンチャー をクリックします
  • ファイル選択画面にて、空のフォルダを作成して、決定ボタンを押します

使い方 (シンプルな方法)

  • game.exe を sample/ の中にコピーします
  • sample/ の中で game.exe を起動します
